# 🌤 Weather Prediction using Machine Learning

A simple Machine Learning web application built using **Python, Flask, HTML, CSS, and Scikit-Learn** that predicts the temperature based on weather parameters.

---

## 📌 Features

- Predicts Temperature (°C)
- Beautiful Weather UI
- Input Validation
- Responsive Design
- Machine Learning Model
- Weather Condition Display

---

## 📊 Input Parameters

- 💧 Humidity
- 🌡 Pressure (millibars)
- 🌬 Wind Speed (km/h)

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Machine Learning
- Pandas
- NumPy
- Scikit-Learn
- Pickle

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/weather-prediction.git
```

Go to the project folder

```bash
cd weather-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
